Kebijakan Privasi

Jualan Cepat · Diperbarui: 1 Juni 2026

Ringkasan

Jualan Cepat adalah aplikasi POS offline. Data toko Anda disimpan di perangkat ini, bukan di server kami. Kami tidak menjual data Anda kepada pihak ketiga.

Data yang disimpan

Aplikasi menyimpan data berikut secara lokal di perangkat Anda:

  • Profil toko (nama, alamat, telepon)
  • Produk, kategori, dan harga
  • Transaksi penjualan, pembayaran, hutang, dan refund
  • Nama/telepon pelanggan (jika Anda mengisi saat penjualan hutang)
  • Preferensi app (mis. status onboarding, waktu backup terakhir)

Data yang tidak kami kumpulkan

Tidak ada akun login, tidak ada analytics pihak ketiga, dan tidak ada pengiriman data transaksi ke cloud secara otomatis. Backup hanya terjadi jika Anda mengekspor file backup secara manual.

Izin perangkat

Mikrofon — hanya untuk fitur pencarian produk suara (opsional). Anda bisa menolak izin; fitur lain tetap berfungsi.

Backup

File backup (.jualancepat) dienkripsi dengan PIN yang Anda buat. Anda bertanggung jawab menyimpan file dan PIN tersebut. Kami tidak menyimpan salinan backup di server.

Penghapusan data

Jualan Cepat tidak memakai login atau akun cloud. Untuk menghapus semua data pengguna, lakukan salah satu cara berikut:

  1. 1 Buka app → menu drawer → Privasi & data → Hapus semua data di HP ini
  2. 2 Atau uninstall aplikasi / Hapus data di Pengaturan Android untuk Jualan Cepat

Setelah dihapus, data tidak dapat dipulihkan kecuali Anda punya file backup yang diekspor sebelumnya.
